# Д/З 1

Реализовать для функции

$$ \sin(\text{ф} x) + e^{\text{и} x} + \text{ф} x^2 + \text{и} x + \text{о} + x^4, $$

где $\text{ф}, \text{и}, \text{о}$ --- количество букв в фамилии, имени и отчестве
соответственно, следующие методы:
1. пассивного поиска,
1. дихотомии,
1. золотого сечения,
1. касательных,
1. Ньютона-Рафсона,
1. секущих.
